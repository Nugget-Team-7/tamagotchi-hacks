Features

  • Open: Click 'Choose File' to open a Smacard dump file directly on your computer. No upload is neccesary, all operations run locally.
  • Reset Usages: Click the 'Clear' buttons to reset Device ID usages.
  • Download Images: Right-click an image to download it.
  • Replace Images:
    1. Click on an image to replace it.
    2. Open your .png or .bmp image. Your image should contain the same number or fewer colors than the palette size for the image you are trying to replace. For best results, use an image that is the same size as the image you are replacing.
    3. For image sets with only one image you can just click the 'Save' button.
    4. For sets with multiple images using different colors, you may need to preserve palette colors as you import subsequent images. Check the boxes of colors that you need to preserve from the previous image(s).
    5. Note that some image sets share a palette with other image sets and you may need to use the checkboxes in this case as well.
  • Export: Click 'Export' button to export your Smacard file with all modifications.
  • Open-source: You can save the entire site to your Desktop and use it there. Feel free to use the code as you see fit.
